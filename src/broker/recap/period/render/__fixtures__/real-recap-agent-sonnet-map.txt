  - Group-level virtualization is ineffective; 500 entries collapse to ~8 huge groups rendered whole

  - resumeConversation unconditionally stamps lastActivity on every meta arrival

  - OpenRouter real billed cost requires usage:{include:true}; commits were never seen by recap LLM (stubbed git gather)
  - Biome formatter is more stable than oxc; oxlint has a11y/hooks gaps
  - Voice recordings target the currently-selected conversation, not the one recorded against

## TL;DR
Two large foundational shifts landed this week: **profile was removed from project URIs** (now `LaunchConfig.sentinelProfile` metadata) and **daemon was reframed from a backend to a transport** (`transport:'claude-daemon'`). The whole stack must stay consistent on both — `daemon://` URIs and `backend:'daemon'` are dead. Most transcript-perf, conversation-flapping, and daemon-identity bugs are fixed and deployed; recap v2.1 chunked map-reduce shipped.

## State
- **Profile-URL-strip is live** (c8704b8d). Project URIs no longer carry `profile@`; profile rides in `LaunchConfig.sentinelProfile`. 67 legacy `claude:///` settings keys were deleted, not migrated. Watch for any code still reading profile from a URI.
- **Daemon is a transport, not a backend.** `transport:'claude-daemon'` replaced `backend:'daemon'`; `daemon://` URIs are normalized to `claude://` (2427318b). A boundary-lint rule guards `transportMeta` reads. Persisted profiles may still carry `backend:'daemon'` shape — UI derives transport from `(backend, headless)`.
- **Conversation flapping is fixed** (78d735cb): the phantom reaper now skips daemon conversations (they have no agent socket by design); orphaned-socket eviction now closes sockets explicitly (621f6a69); ccSession cross-wiring fixed via pid+lsof correlation (3745d1f9).
- **Transcript perf**: `key=` remount dropped (5803c615); stable group IDs prevent remounts on tail-append (9dfcad57); index-based scroll anchor fixes scrollback jerk (4cac6c0f). Progressive/infinite scrollback and events-on-demand shipped. Frontend-health chain merged to main (react-doctor 59→86, a2bb5d21).
- **Recap v2.1 shipped**: chunked map-reduce (Sonnet map → code merge → Opus synthesis, 1fd03fe5), agent-triggered recaps, MCP tuning params, real OpenRouter cost via `usage:{include:true}`, output limit raised 8k→32k. Pillars C+/C++/E/F (bundle/regenerate/cost analytics/retrospect) in ac4dfd98.
- **Reaper cascade-delete is live** (49f89874) with a **90-day TTL** — its first run will purge old conversations. 645 pre-existing orphaned conversations (208,937 entries) remain invisible; no recovery tool built yet.
- [inferred] Many parallel worktrees were in flight across multiple agents; 8 merged branches were cleaned up but expect dirty shared worktrees and live processes belonging to other agents.

## Decisions
- **Profile belongs in launch metadata, not URIs** (4a0318fc) — it violated URI identity semantics and broke permission/grouping; done as a hard cutover with no deprecation window, requiring a full 4-component deploy.
- **Daemon uses the `claude://` scheme** (5a7e6074) — transport differences live in `agentHostType`, not the URI, so the same physical project stays in one identity bucket.
- **Delete legacy settings rather than migrate** (52835091) — kill the empty-authority `claude:///` keys outright.
- **Always fetch on conversation switch** (7389976a) — the HIT cache optimization showed partial transcripts from WS pushes; correctness over the saved round-trip.
- **90-day cascade TTL over forever-retention** (b60edb95) — bounded in-memory mirror with clean cascade delete beats a complex disk-fallback cache.
- **Sonnet map + Opus reduce, JSON-only intermediaries** (389972df) — cheap structured extraction, expensive narrative only on the small merged input.
- **Stay on Biome, reject oxc** (87f16e52) — oxfmt is 0.x beta, oxlint lacks a11y/hooks rules in use, and the timing was hostile with many active worktrees.
- **Adopt Harness patterns, replace its runtime** (5194ff48) — keep the 6-pattern vocabulary but use `spawn_conversation`/`send_message`.

## Dead ends — do NOT retry
- **Agent host does NOT make HTTP calls to broker `/mcp`** (807269d9) — all spawns go via WS `channel_spawn`. There is no outbound MCP HTTP client. Note `dispatchClaudeDaemon` does not call `addRendezvous`, which breaks caller tracking for daemon spawns.
- **Group-level virtualization for transcript perf** (576c695f, 3860a533) — 500 entries collapse into ~8 huge groups rendered whole, so it does nothing. Cost scales with DOM nodes, not entries.
- [inferred] **Longtask observer on Safari** (3860a533) — Safari supports neither `longtask` nor LoAF; it was dead weight.
- [inferred] **Reading CC's `CwdChanged event.data.cwd`** (c9ca9297) — the field is fine; the real bug was that `currentPath` was never serialized to the UI DTO.
- **Git evidence binding via `git cherry` / from broker shell** (fecbb90a, 1f0bd0ba) — broker runs in Docker without host filesystem access, and `git cherry` lies on squash/rebase merges. Use `merge-tree` no-op tests for integration detection.

## Gotchas
- **Biome `--fix` rewrites the whole repo.** `bun run lint` runs `biome check --fix .` and reformats 14-48 unrelated files, creating uncommitted noise that blocks merges and causes rebase conflicts. Stage only your own files.
- **Zustand object-returning selectors cause React #185 infinite loops** — they fail `Object.is`. Use per-field selectors. This crashed the entire control-panel header once (e34f87a4).
- **React `key=` change forces a full subtree remount**, bypassing all memoization. Edit-diff re-renders were caused by virtualizer keys on tail seq remounting, not memo misses.
- **`Popover.Trigger asChild` requires forwardRef + prop spread** on wrapper components or the floating-UI anchor breaks (a1b1f4c2).
- **FTS5 treats hyphens as NOT** — `war-council` becomes `war NOT council`. Token-level quoting is the fix (efdd346f).
- **Worktree tool-path trap**: shell `cd` does not redirect Edit/Write tool paths — use absolute worktree paths. Worktrees also need symlinked `node_modules` for typecheck/build.
- **`bun test` only runs the server suite** (bunfig `root='src'`) — run web vitest separately or miss frontend regressions.
- **PWA caching**: the control panel is a PWA; soft reload serves the cached bundle. Hard refresh after broker/web deploys.
- **Broker changes need a full Docker recreate and fleet bounce**; web-only changes deploy instantly via volume mount on `web/dist`.
- [inferred] **MCP spawn rendezvous timeout fires too eagerly** for daemon conversations that boot fine (7370a1ee). Agent-host role is silently rejected from `recap_create`/`rename_conversation` (requestId dropped → 30s timeout instead of an error).
- **`ensureCanonical` mutates tool input in place** (74e24505), breaking shared module-level test objects across renders.
